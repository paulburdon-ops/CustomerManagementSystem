# Customer Management System

## Overview

This project is part of my Python learning workshop.

The application demonstrates:

- Python modules
- Functions
- Dataclasses
- SQLite
- Context managers
- Type hints

## Project Structure

main.py
- Application entry point.

customer_storage.py
- Handles all SQLite database operations.

customer_functions.py
- Legacy JSON implementation for comparison.

models.py
- Contains the Customer dataclass.

customers.db
- SQLite database.

## Current Features

- Create customer table
- Add customer
- List customers
- Find customer by account
